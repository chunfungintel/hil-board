HIL Arduino Installation & Flashing

Package content:
    - hil_arduino_install.bat                           - The script used to install the Arduino drivers
    - hil_arduino_flash.bat                             - The script used to flash the Arduino f/w
    - .\drivers                                         - The drivers for the Arduino board 
    - utils

1. Install the Arduino drivers
Run the script 'hil_arduino_install.bat [noprompt]'
Note: Run the script only in Administrator mode!

2. Flash the Arduino board
RUn hil_arduino_flash.bat [noprompt]

Please contact harts_support@intel.com for any issues regarding flashing of the Arduino board.
